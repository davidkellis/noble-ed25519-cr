# noble-ed25519

This is a crystal port of https://github.com/paulmillr/noble-ed25519.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     noble-ed25519:
       github: davidkellis/noble-ed25519
   ```

2. Run `shards install`
